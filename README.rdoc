Welcome to SurfBuddy (still working on name)


This is an application designed for surfers to find other surfers to go surfing with.

Users will be able to login, make posts about where/when they want to go, comment on other users posts, and send 1-1 messages with other users.

There will be custom weather and surf reports to reference, and user profiles to view. 

May also try to include some surf event's and sweet graphs.
